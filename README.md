
## BackEnd (NestJS):

```bash
  cd server
  yarn
  yarn start:dev
```

**Remember to change server env vars**

> **.env.example**

    NODE_ENV=development
    ALLOWED_ORIGIN=
    DB_NAME=
    DB_USERNAME=
    DB_PASSWORD=
    LANG=pt_BR.UTF-8

## App (RN):

```bash
  cd client
  yarn
  yarn android|ios
```
